# Final Project Summary

This project predicts student final grade `G3` using regression machine learning models.

## Dataset
- Math dataset: (395, 33)
- Portuguese dataset: (649, 33)
- Combined dataset: (1044, 33)

## Best Result
Best model: **Gradient Boosting Regressor**  
RMSE: **1.5410**  
R²: **0.8464**

## Conclusion
The best model performed well because it captured nonlinear relationships between previous grades, study behavior, absences, failures, and final grade. Previous grades were the strongest predictors.
